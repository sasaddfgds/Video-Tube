# Проект: Video-Tube (Клон YouTube для техникума)

**Цель:** Разработать быстрый, качественный и уникальный видеохостинг (без полного копирования дизайна YouTube).
**Язык интерфейса:** Польский (Polski).
**Ожидаемая оценка:** Минимум 3 балла (с потенциалом на большее за счет интересных "фич").

## 🛠 Технологический стек
* **Front-end:** HTML5, CSS3, Vanilla JavaScript.
* **Back-end:** PHP (чистый, без фреймворков).
* **База данных:** SQLite (легкая, не требует настройки сервера).

## 💡 Идеи, чтобы "удивить" учителя (Киллер-фичи)
Чтобы получить хорошую оценку, не усложняя проект до бесконечности, реализуем следующие вещи:
1. **Кастомный видеоплеер (Custom HTML5 Player):** Вместо стандартного плеера браузера сделаем свои стилизованные кнопки (Play/Pause, громкость, прогресс-бар) на Vanilla JS. Это покажет отличное владение JavaScript.
2. **Асинхронные запросы (Fetch API):** Лайки под видео и добавление комментариев без перезагрузки страницы. Это продемонстрирует отличную связку JS + PHP.
3. **Темная/Светлая тема (Dark Mode):** Переключатель тем с сохранением в `localStorage`. Очень эффектно выглядит на защите.
4. **"Живой" поиск (Live Search) с паттерном Debounce:** Поиск роликов по названию без перезагрузки страницы. Чтобы показать профессиональный подход, в JS будет реализован `debounce` — запросы к БД пойдут только когда пользователь закончит ввод (пауза 300мс), чтобы не нагружать сервер. Преподаватели очень ценят понимание таких оптимизаций.
5. **Плавающий мини-плеер (Sticky Player):** При прокрутке страницы вниз (чтобы почитать комментарии) видео плавно уменьшается и закрепляется в углу экрана. Отличная демонстрация продвинутой работы с CSS (анимации, fixed position) и JS (Intersection Observer или события скролла).
6. **One-Click Local Run:** Специальный `.bat` скрипт для мгновенного запуска встроенного PHP-сервера. Не требует настройки Apache/XAMPP/MySQL. Учитель сможет запустить проект двойным кликом.

## 📌 Базовый функционал (MVP)
* **Главная страница:** Сетка видеороликов (миниатюра, название, автор, дата).
* **Страница просмотра видео:** Сам плеер, описание, секция комментариев и лайки.
* **Панель загрузки:** Форма для загрузки видеофайла и постера (обложки) на сервер через PHP.
* **Система пользователей (базовая):** Регистрация и авторизация (с хэшированием паролей).

## 🚀 План разработки
- [ ] **Шаг 1:** Проектирование БД (SQLite) - таблицы пользователей, видео, комментариев.

- [ ] **Шаг 3:** Написание PHP-логики (регистрация, загрузка файлов, API для лайков).
- [ ] **Шаг 4:** Оживление интерфейса на JS (кастомный плеер, асинхронные запросы).
- [ ] **Шаг 5:** Перевод интерфейса на польский язык и финальное тестирование.
